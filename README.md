(FR) ECE HEROES est un jeu de type match-3, exécutable dans le terminal Windows (PowerShell) ou Linux.  
Compatibilité : UNIQUEMENT LINUX. Pour une machine Windows, veuillez cliquer [ici](https://github.com/Tom-ing/Porjet-info).  
CMake doit être installé et utilisé pour pouvoir compiler le jeu sur la machine.

(EN) ECE HEROES is a project developed by my team. We had to create a Match-3 style game from scratch, playable in the terminal (Windows PowerShell or Linux terminal).  
Compatibility: LINUX ONLY. If you are using a windows computer, please click [here](https://github.com/Tom-ing/Porjet-info). CMake must be installed and used to compile the game.
